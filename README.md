# 🚀 WebChatPlatform

An AI-powered web chat platform built with **Next.js**, featuring intelligent chatbot capabilities, document processing, and real-time analytics.

Designed for modern applications that require conversational AI, structured data handling, and performance insights — all in one scalable platform.

---

## ✨ Features

### 🤖 AI Chatbot (RAG Ready)
- Real-time AI-powered responses  
- Context-aware conversations  
- Easily integrable with LLM providers  

### 📄 Document Processing
- Upload and process documents  
- Extract meaningful insights  
- Use documents as chatbot knowledge sources  

### 📊 Analytics Dashboard
- Track user interactions  
- Monitor conversation trends  
- Analyze engagement metrics  

### ⚡ Modern Stack
- Built with Next.js (App Router)  
- TypeScript support  
- Fast refresh development  
- Optimized fonts with `next/font`  
- Production-ready architecture  

---

## 🏗 Tech Stack

- **Frontend:** Next.js 14+
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **AI Integration:** OpenRouter / OpenAI compatible APIs
- **Deployment:** Vercel-ready

## 🖥 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/Webchatplatform.git
cd Webchatplatform
````

### 2️⃣ Install Dependencies

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

### 3️⃣ Run Development Server

```bash
npm run dev
```

Now open:

👉 [http://localhost:3000](http://localhost:3000)

The app will automatically reload as you edit files.

## 🔐 Environment Variables

Create a `.env.local` file in the root directory:

```
NEXT_PUBLIC_API_URL=
LLM_API_KEY=
DATABASE_URL=
```

Add your AI provider keys and backend configuration here.
